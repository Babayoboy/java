## 🎮 Project Portfolio

This repository contains various mini-projects and games implemented in Java to practice core concepts.

### 🕹️ Games
*   **Hangman**: Classic word guessing game.
*   **Number Guesser**: A game to guess the secret number.
*   **Quiz Game**: Test your knowledge with multiple-choice questions.
*   **Rock-Paper-Scissors**: The classic hand game against the computer.
*   **Slot Machine**: A simple casino-style slot machine simulator.
*   **Dice Roller**: Simulate rolling virtual dice.
*   **Mad Libs**: A fun word-replacement game.

### 🛠️ Tools & Utilities
*   **Banking Programs**: Simulated banking operations.
*   **Calculators**: Basic Calculator & Compound Interest Calculator.
*   **Converters**: Temperature Converter & Weight Converter.
*   **Multimedia**: **Song Player** with playback controls (WAV support).
*   **Productivity**: Countdown Timer & Shopping Cart.
